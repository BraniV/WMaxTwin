# WMaxTwin
Three Exercises for WMaxTwin Paper: Multiscale MaxTwin for Data Splitting as Multiscale Feature Geometry

Data splitting is often treated as a routine preprocessing step, but in validation and cross-validation it is a statistical design problem: the split determines which tuning parameters are selected and how difficult the validation problem is.  Recent deterministic methods such as SPlit, twinning, and MaxTwin improve on random splitting by constructing validation sets that are representative of the data distribution while avoiding excessive redundancy with the training set.  This paper extends that principle to data whose predictive structure is multiresolution.  We propose Multiscale MaxTwin, or WMaxTwin, in which the MaxTwin criterion is applied after a scientifically chosen multiscale feature map.  The central idea is: MaxTwin + multiscale feature geometry.

The contribution is not arbitrary splitting followed by an ordinary discrete wavelet transform.  Rather, wavelets and related multiscale systems define the geometry in which representativeness and training-validation separation are measured.

We develop the mathematical framework, including multiscale energy distance, multiscale separation, and a theoretical clarification that a full unweighted orthogonal wavelet transform cannot change Euclidean MaxTwin.  Hence WMaxTwin becomes genuinely different only through scale weighting, truncation, redundancy, nonlinear summaries, graph or diffusion geometry, or wavelet evaluations at sample locations.  The framework covers whole-object functional observations, spectra, images, spatial fields, scalar density samples, and carefully formulated gridded-signal problems.

Three numerical examples illustrate the method and they are provided in this repository as Jupyter (Python and R), as *.py and *.Rmd  and also as MATLAB.  In claw-density estimation (Claw Directory), Haar scale-location features improve split balance for narrow density components.  In Doppler wavelet shrinkage (Nason Directory), WMaxTwin is combined with a masked wavelet fit and a high-frequency threshold ramp.  In Tecator spectroscopy
(Tecator Directory), a nested WMaxTwin path shows that moderate wavelet perturbations of raw MaxTwin can stabilize functional-regression tuning and improve external prediction.  The results support a cautious but useful conclusion: WMaxTwin is not universally superior to MaxTwin, but it is better targeted when validation difficulty is governed by scale-local structure.
